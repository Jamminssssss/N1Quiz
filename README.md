# jlpt N1 일본어 단어 학습을 위한 퀴즈앱

### 주요기능
1.N1 단어공부

2.오답을 체크하면 빨간불

3.정답을 체크하면 녹색불

4.퀴즈를 다풀면 맞춘갯수와 틀린갯수표시
